# Jankesowa Pasieka 🐝

Elegancki sklep internetowy dla rodzinnej pasieki na terenach nadwiślańskich Kujaw (Topolno, gmina Pruszcz).

## Technologie

- **Next.js 14** (App Router)
- **TypeScript**
- **Tailwind CSS**
- **shadcn/ui** (własne komponenty inspirowane estetyką)
- **Framer Motion** (płynne animacje koszyka)
- **Sonner** (piękne powiadomienia)
- **Lucide React** (ikony)

## Kolorystyka marki

- `#166534` — głęboka zieleń
- `#92400E` — brąz miodowy
- `#F5EDE4` — kremowy
- `#D97706` — złoty akcent

## Funkcjonalności

- Pełna responsywność (mobile-first)
- Koszyk oparty na localStorage + Context
- Edycja ilości w koszyku
- Symulacja składania zamówienia
- Piękne, naturalne zdjęcia produktów (Unsplash)
- Formularz kontaktowy (symulacja)
- Elegancka nawigacja + wysuwany koszyk

## Uruchomienie projektu

```bash
npm install
npm run dev
```

Otwórz [http://localhost:3000](http://localhost:3000).

## Strony

- `/` — Strona główna z hero i wyróżnionymi produktami
- `/produkty` — Pełna oferta z filtrowaniem
- `/koszyk` — Dedykowana strona koszyka
- `/o-nas` — O pasiece i nasze zasady
- `/kontakt` — Formularz kontaktowy

## Uwagi

Projekt nie zawiera prawdziwego backendu ani płatności. 
Zamówienia są symulowane z użyciem localStorage i toastów.
